# Harvextro Backend System

Backend logic for the Harvextro robotic chilli sorting mobile application. 
This system handles robot control, chilli data processing, and summary management using Firebase.



## Project Overview

The Harvextro backend system supports the mobile application used to control 
a robotic arm that sorts Scotch Bonnet chillies based on ripeness.

The backend is responsible for:
- Robot command processing
- Chilli sorting data management
- Summary generation
- Firebase database communication


## Project Structure

src/..

- auth          # Authentication logic (login, register, logout)
- robot         # Robot control commands (start, stop, pause)
- data          # Chilli data processing and summary
- navigation    # App navigation system
- utils         # Helper functions
- firebase.js   # Firebase configuration
